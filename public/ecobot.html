Introduction

ECOBOT is an innovative AI vision-based robotic arm specifically designed to assist stroke patients in their daily activities. By combining advanced computer vision technology with affordable hardware, ECOBOT provides an accessible solution for object manipulation and grasping tasks.

Our project leverages the power of Google's Gemini Vision API to identify and locate objects, while utilizing inverse kinematics algorithms to calculate precise movements for the robotic arm. The entire system is built with cost-effectiveness in mind, making it accessible to a broader range of patients who need assistance.

How It Works

ECOBOT operates through a sophisticated integration of AI vision technology and precise mechanical control, providing seamless object manipulation assistance for stroke patients in their daily activities.

1

Vision Detection

Gemini Vision API analyzes the workspace environment and accurately identifies objects that need to be grasped or manipulated by the robotic arm.

2

Position Calculation

Advanced inverse kinematics algorithms calculate the optimal path and precise servo angles required to reach the target object safely and efficiently.

3

Command Transmission

Python script processes the calculations and sends precise servo angle commands through PySerial communication to the ESP32 microcontroller.

4

Arm Movement

ESP32 microcontroller controls MG996 servo motors to execute the calculated movements and grasp the target object safely and accurately.
